LunaSea

The modern usenet manager for iOS.

 GitHub Downloadv0.3.1+1

Requires iOS 11+, iOS 13 recommended

For and only for iOS.

An iOS usenet manager focused on providing a clean & simple experience for all of your management needs. LunaSea is built to be a modern iOS application, using the Flutter framework.

Automation
LunaSea ships with support for fully controlling your automation tools, including Lidarr, Radarr, and Sonarr.

You can view metadata details, add new content, edit content, search for releases, and more!
Clients
LunaSea ships with support for SABnzbd, and will later support popular usenet & torrent downloading clients, including NZBget, rTorrent, Deluge, and qBittorrent.

You can view and manage your live-updating queue, view your download history, add new torrent & NZB files directly through LunaSea, and more!
Monitoring
(Coming Later)
LunaSea will later ship with support for popular server monitoring tools, including Tautulli, Netdata, Plex, Emby, and Jellyfin.

You can view and manage Plex activity, CPU usage, memory usage, network utilization, and more!

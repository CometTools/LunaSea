LunaSea

A Modern Self-Hosted
Media Manager

 App Storev4.0.0 Play Storev4.0.0

Requires iOS 11+, Android 7.0+

Designed for mobile devices.

LunaSea is a mobile self-hosted media manager focused on providing a clean & simple experience for all of your management needs. LunaSea is built to be a modern mobile application, utilizing the Flutter framework.

Automation
LunaSea ships with support for controlling your automation tools, including Lidarr, Radarr, and Sonarr.

You can view metadata details, add new content, edit content, search for releases, and more!
Clients
LunaSea ships with support for SABnzbd & NZBGet, and also supports directly searching Newznab-based indexers!

You can view and manage your live-updating queue, view your download history, add new NZB files directly through LunaSea, and more!
Monitoring
LunaSea ships with support for popular Plex server monitoring tool, Tautulli.

You can view and manage Plex activity, users, interact with stream data graphs, view user lists, and more!
